# 要件定義書：ドレミ音当てゲーム

**プロジェクト名**：ドレミ音当てゲーム  
**クライアント**：ピアノ＆リトミック教室 Waltz  
**作成日**：2026-04-26

---

## 1. 概要

Web ブラウザで動作するピアノの音当て練習アプリ。音を聞いて正しい音名を答えることで、絶対音感・和音認識を鍛える。サーバー不要の単一 HTML ファイルで構成される。

---

## 2. 対象ユーザー

- ピアノ教室の生徒（子ども〜大人）
- 音感トレーニングを行いたい人

---

## 3. 機能要件

### 3.1 難易度選択

| 難易度 | 対象音 | 問題形式 |
|--------|--------|----------|
| 初級 | 白鍵7音（ド・レ・ミ・ファ・ソ・ラ・シ） | 単音当て |
| 中級 | 全12音（半音含む） | 単音当て |
| 上級 | 全12音（半音含む） | メジャーコード3和音当て |

- 難易度切り替え時にスコアおよび各音の成績をリセットする

### 3.2 音の再生

- 「音を聞く」ボタン押下で音を再生する
- Web Audio API を使用し、倍音（ハーモニクス）を含むピアノ音を合成する
- 上級モードではメジャーコード（根音・長3度・完全5度）を同時再生する
- 再生中は再度の再生を防ぐ（ボタン無効化）
- 回答後に次の問題を自動再生する（2秒後）

### 3.3 回答入力

- 音名ボタンを押して回答音を選択する
- 初級・中級：1音のみ選択可能
- 上級：最大3音まで複数選択可能
- 「回答する」ボタンで送信する

### 3.4 正誤判定

| 結果 | 表示 | 効果音 |
|------|------|--------|
| 正解 | 「✨ せいかい！」＋バウンスアニメーション | 上昇音3音 |
| 不正解 | 「❌ ざんねん。正解は〇〇でした」＋シェイクアニメーション | 低音2音 |

- 正解ボタンをハイライト表示する
- 上級モードでは根音名も正解フィードバックに表示する

### 3.5 スコア表示

- 正解数・不正解数・正答率（%）をリアルタイムで表示する
- 正答率は小数点以下を四捨五入して整数で表示する

### 3.6 各音の成績表示

- 音名ごとに正解数・不正解数・正答率を一覧表示する
- 表示対象は選択中の難易度の音のみ

---

## 4. 非機能要件

| 項目 | 内容 |
|------|------|
| 動作環境 | モダンブラウザ（Web Audio API 対応） |
| 構成 | 単一 HTML ファイル（サーバー不要） |
| レスポンシブ | スマートフォン対応（最小幅 480px 以下で最適化） |
| キャッシュ | `no-cache` 設定により常に最新版を読み込む |
| 外部依存 | なし（ライブラリ・CDN 不使用） |

## 6. 音声仕様

- 音源：Web Audio API によるリアルタイム合成（サイン波 + 倍音）
- 倍音構成：基音 40%、第2倍音 20%、第3倍音 10%、第4倍音 8%
- 再生時間：1.5秒（アタック 50ms、サステイン後にフェードアウト）
